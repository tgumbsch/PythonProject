# PythonProject
A generic Python project template that computes the cummulative sum in Python, R and C++   

## Usage  

```
chmod +x ./Run.sh
./Run.sh
```   

## Requirements

- `Python` and the `pandas` package. 
- `R` and the `xts` package   
- `gcc` version 6.0 or higher and `googletest`
